# Discuss the scope of var, let, and const

1. Variables declared with the var keyword have function scope or global scope, depending on where they are declared.

2. Variables declared with the let keyword have block scope, which means they are accessible only within the block, statement, or curly braces {} in which they are defined.

3. Variables declared with the const keyword have block scope, just like variables declared with let.
